# BizFin SEO Article Generator

Универсальный плагин для генерации SEO-оптимизированных статей с использованием AI-агента и настраиваемой матрицей параметров.

## Описание

BizFin SEO Article Generator — это мощный инструмент для создания высококачественных, SEO-оптимизированных статей для сайта bizfin-pro.ru. Плагин интегрируется с существующей архитектурой блога и предоставляет гибридную модель работы с AI-агентом.

## Основные возможности

### 1. Система динамических модулей
- **Калькулятор банковских гарантий** - интерактивный калькулятор для расчета стоимости
- **Схема-диаграмма процесса** - визуальная схема процесса получения гарантии
- **Сравнительная таблица** - таблица сравнения банков и их условий
- **Актуальные ставки** - блок с актуальными ставками банков
- **Чек-лист документов** - интерактивный список необходимых документов
- **Временная шкала** - timeline процесса получения гарантии
- **Разбор стоимости** - детальный разбор стоимости гарантии
- **Рейтинг банков** - рейтинг банков по надежности и условиям

### 2. SEO-оптимизация
- Автоматическая оптимизация по Yoast SEO
- Интеграция с Alphabet Blog Panel
- Соблюдение всех SEO-требований из матрицы
- Минимальный объем статьи 2500 слов

### 3. Гибридная модель работы
- AI-агент генерирует статьи в чате
- Плагин публикует статьи на сайте
- Полная интеграция с архитектурой блога
- Отслеживание процесса в реальном времени

## Установка и настройка

### 1. Активация плагина
1. Загрузите плагин в папку `/wp-content/plugins/`
2. Активируйте плагин в админ-панели WordPress
3. Перейдите в меню "SEO Article Gen"

### 2. Настройка интеграций
Плагин автоматически интегрируется с:
- Yoast SEO
- Alphabet Blog Panel
- ABP Article Quality Monitor
- ABP Image Generator
- Elementor
- BFCalc Live Rates

### 3. Настройка матрицы SEO
Матрица SEO-параметров настраивается в файле `includes/ai-agent-integration.php` в методе `get_seo_matrix()`.

## Использование

### 1. Создание статьи с модулями
1. Перейдите в меню "Модули"
2. Заполните форму:
   - Ключевое слово
   - Инструкция для статьи
   - Оглавление статьи
   - Выберите модули
3. Нажмите "Сгенерировать статью"

### 2. Тестирование
Используйте кнопку "Тест с модулями" для генерации тестовой статьи "Что такое банковская гарантия".

### 3. Публикация
После генерации статьи вы можете:
- Опубликовать статью
- Редактировать в WordPress редакторе
- Посмотреть результат

## API и интеграция

### AJAX endpoints
- `bsag_generate_with_modules` - генерация статьи с модулями
- `bsag_publish_article` - публикация статьи
- `bsag_generate_test_article` - генерация тестовой статьи

### JavaScript API
```javascript
// Генерация статьи с модулями
BizFinSEOGenerator.generateWithModules(keyword, userInstruction, tableOfContents, modules);

// Публикация статьи
BizFinSEOGenerator.publishArticle(postId);

// Тестирование с модулями
BizFinSEOGenerator.testWithModules();
```

## SEO-матрица

Плагин использует комплексную SEO-матрицу, включающую:

### 1. Замысел и интент
- Единый фокус-ключ
- Интент запроса (информационный/коммерческий/смешанный)
- E-E-A-T (автор, опыт, ссылки на нормативку)

### 2. Метаданные
- Title (50-60 символов)
- Meta Description (120-160 символов)
- URL slug
- H1 (один на странице)
- Open Graph / Twitter

### 3. Разметка Schema.org
- Article/BlogPosting
- BreadcrumbList
- FAQPage (при наличии)
- HowTo (при необходимости)

### 4. Контент и структура
- Первая экранная зона
- Оглавление с якорями
- Глубина проработки
- Внутренние и внешние ссылки

### 5. Изображения и медиа
- WebP/AVIF формат
- Оптимизированные размеры
- Alt-тексты
- Lazy loading

### 6. Core Web Vitals
- LCP ≤ 2.5 с
- CLS ≤ 0.1
- INP ≤ 200 мс
